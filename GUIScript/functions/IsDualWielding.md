**Prototype:** GemRB.IsDualWielding (globalID)

**Description:** 1 if the actor is dual wielding; 0 otherwise.

**Parameters:**
  * globalID - party ID or global ID of the actor to use

**Return value:** bool

**See also:** [GetCombatDetails](GetCombatDetails.md)
