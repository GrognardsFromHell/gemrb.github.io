**Prototype:** GemRB.SetFeat (globalID, feat, value)

**Description:** Sets a feat. Handles both boolean and numeric fields.

**Parameters:** 
  * globalID - party ID or global ID of the actor to use
  * feat - feat index from ie_feats.py
  * value - target feat value

**Return value:** N/A

**See also:** [HasFeat](HasFeat.md)
